# Synthetic PPG Data Simulator

Offline desktop tool for generating **fully synthetic, non-clinical PPG waveforms** for ML prototyping, demos, and internal experimentation.

**No patient data · Runs locally · Not for clinical or regulatory use**

---

## What this tool is
- Generates synthetic photoplethysmography (PPG) waveforms
- Designed for ML prototyping, signal-processing pipelines, and product demos
- Preset-based to avoid the need for clinical expertise
- Runs fully offline on your machine

## What this tool is NOT
- Not clinical-grade
- Not medically validated
- Not suitable for diagnosis, monitoring, or treatment
- Not acceptable for regulatory submissions

---

## How it works
1. Launch the desktop application
2. Select a preset or adjust parameters
3. Generate a waveform
4. Export CSV for offline use
